# Custom Domain Notes

BuzzCalculus can run on a custom domain through GitHub Pages.

1. Buy or choose a domain.
2. In the repository, open `Settings -> Pages -> Custom domain`.
3. Enter the domain, for example `buzzcalculus.example.com`.
4. Add the DNS record required by GitHub Pages.
5. After DNS is verified, enable `Enforce HTTPS`.

Do not add a `CNAME` file until the real domain is decided.
