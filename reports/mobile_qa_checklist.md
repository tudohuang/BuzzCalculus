# BuzzCalculus Mobile QA Checklist

日期：2026-06-05  
版本：BuzzCalculus v0.9.0-beta  
Build：2026-06-05

## 測試目標

確認手機版沒有重大操作中斷，尤其是：

- 手寫黑板可以正常書寫、清除、全螢幕、退出
- 選擇題可以正常點選，選項 LaTeX 正常渲染
- 錯題本、每日挑戰、作答歷史、本機紀錄正常
- JSON 匯出 / 匯入不會破壞 localStorage

Beta 公開前，Android Chrome 與 iPhone Safari 都必須完成本表。

## 測試網址

桌機本機：

```text
http://127.0.0.1:3003/index.html?vscode-livepreview=true
```

手機同 Wi-Fi：

```text
http://192.168.50.170:3003/index.html?vscode-livepreview=true
```

如果手機連不上 `192.168.50.170`：

- 確認手機與電腦在同一個 Wi-Fi
- 確認 Live Preview 有對區域網路開放，而不是只綁 `127.0.0.1`
- 確認 Windows 防火牆沒有擋 `3003`
- 必要時改用可綁 `0.0.0.0` 的靜態伺服器測試

## 問題分級

P0：阻止 Beta。流程卡死、無法作答、無法送出、白畫面、資料遺失、手機主要流程不能用。

P1：Beta 前應修。畫面重疊、按鈕很難點、LaTeX 爆版、黑板體驗明顯不順、匯入匯出有邊界問題。

P2：可延後。文案、間距、次要視覺瑕疵、少數題目的提示或標籤微調。
