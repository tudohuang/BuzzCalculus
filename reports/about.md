# About BuzzCalculus

日期：2026-06-05

BuzzCalculus 是一個純前端微積分練習工具，目標是讓使用者在手機與桌機上都能快速練題、手寫草稿、檢查答案，並用本機紀錄追蹤錯題與進步狀態。

## 核心定位

- 不需要帳號
- 不需要後端
- 不需要雲端同步
- 紀錄儲存在本機 `localStorage`
- 適合考前練習、錯題重練、手機碎片時間練題

## 主要功能

- 手寫答題
- 選擇題答題
- 手機黑板與全螢幕黑板
- 答案輸入即時 LaTeX 預覽
- 題目與選項 LaTeX 渲染
- 錯題本
- 黑板草稿回顧
- 提示系統
- 自適應出題
- 每日挑戰
- 作答歷史
- 練習模式
- 本機成就
- JSON 匯出 / 匯入

## 題庫範圍

- 極限
- 微分
- 積分
- 級數
- 多變數微積分
- 進階積分技巧
- 全微分、Hessian、Wronskian、Jacobian
- 複變入門題

## Beta 版本說明

目前版本為 `v0.9.0-beta`。

Beta 階段的重點不是新增大型功能，而是確認：

- 題目與答案正確
- 答案檢查器可靠
- 手機操作穩定
- 部署流程可重複
